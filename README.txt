這個資料夾裡的三種算命程式：

1. 七千單算命：
程式   ：answer_vocab.py  (會從 vocabulary.txt 裡面隨機抽出一個英文單字)
優缺點  ：英文的，如果你希望他的答案是一個詞而不是字的話推薦用這個。
資料來源： https://douze.pixnet.net/blog/post/398218087


2. 電碼算命：
程式   ：answer_code.py  (會從 code.txt 裡面根據電碼隨機抽出一個中文字)
優缺點  ：中文的，但很容易出現生僻字。
資料來源： 周均翰的電碼程式


3. 中文頻率算命：
程式   ：answer_frequency.py  (會從 frequency.txt 裡面根據字的常用度隨機抽出一個中文字)
優缺點  ：中文的，出現的基本上都是常用字，但有3%機率出現「的」。(個人認為3比2優)
資料來源： https://lingua.mtsu.edu/chinese-computing/statistics/char/list.php?Which=TO
