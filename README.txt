這個資料夾裡的三種算命程式：

1. 七千單算命：
answer_vocab.py  (會從 vocabulary.txt 裡面隨機抽出一個英文單字)

2. 電碼算命：
answer_code.py  (會從 code.txt 裡面根據電碼隨機抽出一個中文字)

3. 中文頻率算命：
answer_frequency.py  (會從 frequency.txt 裡面根據字的常用度隨機抽出一個中文字)


優缺點：
1.  英文的，如果你希望他的答案是一個詞而不是字的話推薦用這個。
2.  中文的，但是很容易出現生僻字。
3.  中文的，出現的基本上都是常用字(個人認為3比2優)。
